## ğŸ¯ How to Use

### Creating a Task
1. Click the **Calendar icon** in the bottom navigation to access the Create Task screen
2. Fill in the task details:
   - **Title**: Name of your task
   - **Date**: Due date
   - **Start/End Time**: Time range for the task
   - **Description**: Additional details (optional)
   - **Category**: Select from Marketing, Meeting, Dev, or Design
3. Click **Create Task** to save

### Managing Tasks
- **View Tasks**: Navigate between Home and Calendar views using the bottom navigation
- **Search**: Click the search icon and enter keywords to filter tasks
- **Filter by Tab**: Use "My Tasks", "Project", or "Note" tabs to filter by category
- **Complete Task**: Check the checkbox next to a task in Calendar view
- **Delete Task**: Click the trash icon on any task card

### Customization
- **Toggle Dark Mode**: Click the moon/sun icon in the top-left corner
- **Switch Dates**: Use the horizontal date picker in Calendar view
